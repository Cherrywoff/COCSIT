# COCSIT Backend API

This directory is set up for the backend application servers and security logic.

### Planned Architecture:
* **Server Stack:** Node.js (Express) or Python (Django/FastAPI)
* **Database Driver:** PostgreSQL / MySQL
* **Authentication:** JSON Web Tokens (JWT) mapped to role-based privileges (Student, Faculty, HOD, Super Admin)
* **APIs:** Endpoints for notice boards, timetables, internal grading marks, and attendance logs
