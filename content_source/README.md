# College Information & Content Source Catalog

This directory holds the raw information extracted directly from the active [COCSIT website](https://www.cocsit.org.in/). As we develop the new portal, we can add more contents, texts, and assets here to serve as the unified database source.

## File Directory:

* **[branding.json](file:///c:/Users/radhe/OneDrive/Desktop/COCSIT/content_source/branding.json)**: Institutional details, NAAC accreditations, AISHE codes, address details, and regional sister colleges.
* **[departments_and_staff.json](file:///c:/Users/radhe/OneDrive/Desktop/COCSIT/content_source/departments_and_staff.json)**: Listing of departments, executive administrators, and Heads of Departments (HODs) representing the college hierarchy.
* **[courses.json](file:///c:/Users/radhe/OneDrive/Desktop/COCSIT/content_source/courses.json)**: Full lists of undergraduate (UG) and postgraduate (PG) courses, focus descriptions, and syllabus systems.
* **[placement_partners.json](file:///c:/Users/radhe/OneDrive/Desktop/COCSIT/content_source/placement_partners.json)**: Training and placement cells details, along with major corporate hiring partners.
