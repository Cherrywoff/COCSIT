Conceptual Proposal

COCSIT Rebuild Blueprint Proposal

A strategic plan to build a modern, high-performance, and secure digital ecosystem for the college, using the current site as the informational database.

The Three Pillars of the Ecosystem

1. Public Showroom

The public web pages that require no login access. This serves as the college's digital brochure for families and auditing boards.

  • Dynamic admissions & courses catalog
  • Syllabus, academic calendar documents
  • Clean, modern news notices boards
  • Interactive alumni & placement records

2. Secure Portals

Authenticated dashboards for registered users to manage daily academic operations securely.

  • Student progress tracker dashboard
  • Faculty marks & grading console
  • Subject timetables & schedule views
  • Digital syllabus & assignment submissions

3. Admin Control Room

The unified control panel where authorized administrators publish documents and manage users.

  • Visual content and notice editor
  • User registrations and profile systems
  • Database backup controls
  • System settings & audit logs console

Role & Permission Matrix

User Role Access Type System Permissions & Capabilities
Super Admin College-Wide Complete site modification capabilities, database configurations, employee accounts registration, and full edit controls on all pages.
HOD Departmental Manage department course files, edit specific class schedules, assign professors, and dispatch departmental notifications.
Faculty Course/Class Register classroom attendance percentages, submit student grades and evaluations, and upload class notes.
Student Personal Only View academic attendance track files, check test results, view timelines, and upload project files.
